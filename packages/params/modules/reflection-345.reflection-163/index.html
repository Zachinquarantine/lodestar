    Let ACTIVE_PRESET¶

    • ACTIVE_PRESET: PresetName = parsePresetName(process?.env?.LODESTAR_PRESET as string)

    Defined in activePreset.ts:25

    The preset name currently exported by this library

    The LODESTAR_PRESET environment variable is used to select the active preset If LODESTAR_PRESET is not set, the default is mainnet.

    The active preset can be manually overridden with setActivePreset
